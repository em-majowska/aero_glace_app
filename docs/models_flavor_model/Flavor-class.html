Flavor

Flavor class

Représente un nouveau parfum.

Contient toutes les informations nécessaires pour l'affichage et la gestion d'un parfum dans le catalogue et le panier.

Arguments :

  • id : Identifiant unique du parfum.
  • title : Le nom du parfum.
  • description : La description du parfum.
  • tags : Liste des étiquettes associées au parfum.
  • imagePath : Chemin de l'image du parfum relatif au dossier assets/images/flavors/.
  • meshColors : Couleurs utilisées pour le fond en dégradé maillé du parfum (4 requis).
