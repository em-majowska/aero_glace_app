HiveItem

HiveItem class

Représente un item sauvegardé dans la boîte Hive pour le panier.

Chaque HiveItem correspond à un parfum ajouté au panier avec sa quantité. Cette classe est utilisée pour persister les données du panier entre les sessions.

Arguments :

  • flavorId : identifiant du parfum correspondant.
  • qty : quantité initiale de cet item (par défaut 1).
